PROCEDURE D'INSTALLATION :

A. COTÉ BACKEND
	1/ Cloner ce dépot (se sera le backend)
	2/ Dans le dossier BACKEND :
		npm install
	3/ Copier le fichier .env dans le dossier BACKEND


B. COTÉ FRONTEND
	1/ Cloner le dépot d'Open Classrooms :
		https://github.com/OpenClassrooms-Student-Center/dwj-projet6
	2/ Dans le dossier dwj-projet6 :
		npm install --save-dev node14.0
		npm install --save-dev @angular/cli@7.0.2
		npm install --save-dev sass


POUR LANCER L'APPLI :

A. COTÉ BACKEND
	1/ Dans le dossier BACKEND :
		node server

B. COTÉ FRONTEND
	1/ Dans le dossier dwj-projet6 :
		npm start
